A streamlined Python application built with **Streamlit** and **Google Gemini** that provides clear, line-by-line explanations for code snippets in various programming languages.

##  Overview

AI Code Explainer serves as an interactive educational tool. Users can paste code snippets, select their programming language, and receive a structured explanation generated by Google's Gemini AI. The output includes a high-level summary, a detailed line-by-line breakdown, and a list of key programming concepts used.

##  Features

* **Multi-Language Support:** Analyzes Python, JavaScript, Java, C++, HTML/CSS, SQL, and more.
* **Structured Explanations:** Returns data in a specific format (Summary, Line-by-Line, Key Concepts) for better readability.
* **Secure API Handling:** Users input their own API keys via the sidebar, ensuring security.
* **Clean UI:** Built with Streamlit for a responsive and dark-mode-friendly interface.

##  Tech Stack

* **Frontend/App Framework:** [Streamlit](https://streamlit.io/)
* **AI Model:** [Google Gemini API](https://ai.google.dev/) (`gemini-2.5-flash`)
* **Language:** Python 3.x

##  Usage

1.  Open the app in your browser (usually at `http://localhost:8501`).
2.  Navigate to the **Settings** sidebar and enter your **Google Gemini API Key**.
    * *Need a key? Get one [here](https://aistudio.google.com/app/apikey).*
